# ShadeDemo
一个自定义形状的卡片列表，并且可以修改阴影颜色
#Usage
可以直接引用CardView
```
CardView cardView=new CardView(context);
//修改CardView的主题颜色
cardView.changeTheme(0xff01a3a1);
```
